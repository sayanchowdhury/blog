Migrate a running process into tmux

Being a regular tmux user. Migrating a running process into tmux using reptyr comes handy. reptyr is a utility for taking an existing running program and attaching it to a new terminal. Started a long-running process over ssh, but have to leave and don’t want to interrupt it? Just start a screen, use reptyr to grab it, and then kill the ssh session and head on home.
